## 단축키

**Alt + Q** : 검색창 열고 닫기

**Alt + 2** : 더블콘 ON/OFF

**Alt + B** : 대왕콘 ON/OFF

**TAB** : 다음 콘으로 포커스 이동

**Shift + TAB** : 이전 콘으로 포커스 이동

**상하좌우 방향키** : 포커스 상하좌우로 이동

**Ctrl + 좌우 방향키** : 검색 결과에서 다음 페이지로 이동

**Alt + S(또는 우클릭)** : 콘 즐겨찾기

**Enter(또는 클릭)** : 콘 등록

**Ctrl + Enter(또는 Ctrl + 클릭)** : (더블콘 모드에서) 검색 결과 초기화 하지 않고 첫 번째 콘 선택

**Alt + Shift + D** : 다크모드 ON/OFF
